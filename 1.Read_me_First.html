What is the .claude folder? (For beginners)

The .claude folder is Claude Code's brain — a hidden directory Anthropic's tool creates in your project.

It holds your AI team's "office":

  • agents/ — AI teammates like @chief
  • skills/ — Abilities (git, testing)
  • commands/ — Slash commands (/bootstrap)
  • session-state.json — Memory across sessions
  • CLAUDE.md — Project rules and constitution

Sources:

  • Claude Code Docs
  • GitHub Repo
  • Agents.md Spec

It's where your AI lives and remembers.
